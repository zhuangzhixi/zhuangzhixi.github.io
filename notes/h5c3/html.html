# 1.1.3 Web标准

web标准是由W3C（万维网联盟）所制定的。

为什么需要Web标准？浏览器不同，它们显示页面或者排版有些许差异。

web标准的构成：

结构：结构用于对网页元素进行整理和分类。主要是html

表现：设计网页元素的版式、颜色、大小等外观样式。主要是css

行为：网页模型的定义及交互的编写。主要是javascrip

# 1.2.5.6图像标签：

单标签<img>

<img src="图像URL"/>
1

属性：就是这个图像标签的特性。

属性 属性值 说明
src 图片路径 必须属性
alt 文本 替换文本，图像不能显示时用于替换的文字
title 文本 提示文本，鼠标放上会提示
width 像素（直接填数字） 图像的宽度（只修改宽或高度时，高度等比例修改）
height 像素（直接填数字） 图像的调试（同时修改宽高时，图片可能会变形）
border（一般不在html中用，一般放在css中） 像素（直接填数字） 图像的边框粗细（边框的占用大小不属于图片的大小中）

注：1、标签的属性必须写在标签名后。2、属性不分先后顺序，标签名、属性与属性之间用空格分开。3、属性采取键值对的格式，即key="value"。

相对路径：下一级路径：/ 如imges/baidu.jpg。

​ 上一级路径../ 如../baidu.jpg。跨越多少个上一级就写多少个../

相对路径是/。本地的绝对路径是\。网络的绝对路径是/。

# 1.2.5.7超链接标签：

anchor:锚

<a href="跳转目标，如外部链接：“http://www.qq.com”" target="目标窗口的弹出方式">文本或图像</a>
1

href：指定链接目标的url地址（必须属性）。

target:链接目标的撕开方式。_self为默值，替换当家窗口打开；__blank在新窗口打开。

链接分类：

1、内部链接。就是路径

2、外部链接。如http://www.baidu.com

3、空链接。当还不确定链接目标时用。<a href="#">首页</a>

4、下载链接。形式和内部链接一样，当地址里是一个如.exe或.zip等类型的压缩文件时，点击会直接下载这个文件。

5、网页元素链接。如文本、图像、表格、音频、视频等都可以添加超链接。如：<a href="http://www.baidu.com> <img src="img.jpg"/> </a>

6、锚点链接。快速定位到页面中的某个位置。实现：1、在链接文本的href属性中，设置属性值为#名字，如<a href="#two">第2集</a>。2、找到目标标签，在里面添加id属性=刚才的名字,如：<h3 id="two">第2集介绍</h3>

表格属性：

一般不在html里设置，后面会在css里设置。表格属性写在<table>里去。

属性名 值 描述
align left、center、right 规定表格相对周围元素的对齐方式
border 1或"" 默认""没有边框，1时有边框
cellpadding 像素值 字和单元格边框的距离，默认1像素
cellspacing 像素值 单元格之间的距离，默认2像素
width/height 像素值或百分比 表格的宽高度

合并单元格：

跨行合并：rowspan="合并单元格的个数"，如合并第2、3行的单元格。把上面的单元格作为目标单元格。如目标单元格中为<td rowspan="2"></td>就表格了该单元格为一个跨行的单元格了，大小为之前两个单元格的大小，之前下面被合并的单元格就不用写了。

跨列合并：colspan="合并单元格的个数"。把左侧的单元格作为目标单元格。

# 1.2.5.12表单标签：

表单域：<form>标签

常用属性：

属性 值 作用
action url地址 提交给的服务器的url地址
method get/post 设置表单数据的提交方式，可取get和post
name 名称 指定表单的名称

表单控件（表单元素）：

1、<input>表单元素：单标签，包含一个type属性，根据不同的type值，输入字段拥有很多种形式，用来指定不同的控件类型。

除type属性外，还有其它属性：

属性 值 描述
name 由用户自定义 定义input元素的名称
value 由用户自定义 规定input元素的值
checked checked 规定些input元素首次加载时应当被选中
maxlength 正整数 规定输入字段中字符的最大长度

2、label标签：用于绑定一个表单元素，当点击<label>标签内的文本时，会自动将光标转到对应的表单元素上，来增加用户体验。

<label for="sex">男</label>
<input type="radio" name="sex" id="sex" />//id才是唯一标识
