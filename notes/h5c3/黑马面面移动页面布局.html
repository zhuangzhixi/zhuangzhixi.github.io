# 1.2 代码规范

1. 类名语义化,尽量精短、明确，必须以字母开头命名，且全部字母为小写，单词之间统一使用下划线“_” 连接
2. 类名嵌套层次尽量不超过三层
3. 尽量避免直接使用元素选择器
4. 属性书写顺序
   布局定位属性：display / position / float / clear / visibility / overflow
   尺寸属性：width / height / margin / padding / border / background
   文本属性：color / font / text-decoration / text-align / vertical-align
   其他属性（CSS3）：content / cursor / border-radius / box-shadow / text-shadow
5. 避免使用id选择器
6. 避免使用通配符*和!important

# 2.5 swiper 插件使用

官网地址：https://www.swiper.com.cn/ (opens new window)

  • 下载需要的css和js文件 html页面中 引入相关文件
  • 官网找到类似案例，复制html结构，css样式 js 语法
  • 根据需求定制修改模块

1、引入两个文件.swiper.bundle.mini.css和swiper.bundle.mine.js。

2、找到合适的模板打开新窗口，查看源码，复制相应的swiper结构和样式（除body和html之处的css)和js(最下面的js代码）。可以把js里面的代码放到（function(){}）()里，这样两个轮播图可以区分开，因为函数里面的自变量是局部变量，就不用再去改名字了。

3、点击官网里面的API文档，里面组件里的分布器，里面的pagination，就有教怎么修改pagination的，改的是轮播图里的小圆点。不要分布器时，直接把代码里的pagination这部分删掉就可以了。

4、要想带箭头，去官网里找个带箭头的模板，看下有没有多了啥东西。最后发现是navigation，复制过去就可以了，不仅是js要复制过去，相应的div也要复制过去。

5、然后把每个swiper-slide里面的东西换成一张图片和一行文字。

6、样式.swiper-slide-active就是当前选中的那个slide，中间的那个。

7、修改透明度用opacity:0.6，0-1越来越不模糊。

8、在把swiper-container的宽度改小后，箭头也收缩进去了，要想它在两侧，因为swiper-container有overflow:hidden，会被切掉，所以只能把两个箭头拿出来到swiper-container的外面 。子绝父相，需要在箭头的父亲加个相对定位才能看到。

9、js代码里slidesPerView: 3,表示能够显示的个数。有些被挡住的话，可以写2.几个，根据实际大小来写。

10、js里的spaceBetween: 30,为每个slide之间的距离。

11、centeredSlides: true,把当前最大的放在中间显示。不然它默认是在左边的。

12、如果有多个轮播图，最好修改一个swiper-container的名字，不然会相互影响。

# 2.7 上传码云并发布部署静态网站

准备工作： 需要下载git软件 需要码云注册账号

git 可以把我们的本地网站提交上传到远程仓库（码云 gitee）里面 类似以前的 ftp

码云 就是远程仓库， 类似服务器

  1. 码云创建新的仓库。 heimamm

  2. 利用git 提交 把本地网站提交到 码云新建的仓库里面

    • 在网站根目录右键-- Git Bash Here

    • 如果是第一次利用git提交，请配置好全局选项

      git config --global user.name "用户名"
git config --global user.email "你的邮箱地址"
      1
      2

    • 初始化仓库

      git init
      1

    • 把本地文件放到暂存区或者git add -A也是一样的。

      git add .
      1

    • 把本地文件放到本地仓库里面

      git commit -m '提交黑马面面网站'
      1

    • 链接远程仓库

      git remote add origin 你新建的仓库地址
      1

    • 把本地仓库的文件推送到远程仓库 push

      git push -u origin master
      1

  3. 码云部署发布静态网站

    这里的仓库名字没有要求。

    • 在当前仓库中，点击 “服务” 菜单

  • 选择 Gitee Pages

    • 选择 “启动” 按钮

  • 稍等之后，会拿到地址，就可以利用这个地址来预览网页了如http://xiaoqiang001.gitee.io/heimamm/

  • 所以如果是在github的话，仓库命名为用户名.github.io.仓库名就可以了。

    • 当然你也可以利用 草料二维码 生成二维码 https://cli.im/ (opens new window)

最后： 如果提交网站，你不愿意用git 提交， 可以直接找到仓库，里面有文件，选择上传本地文件即可。

但是，1个小时内，只能上传 20个以内的文件， 前端人员，git必备技能

不想用git提交gitee时

创建仓库时选择初始化仓库，点击创建就可以了。——初始化redme文件——文件（上传文件）。使用这种方式一次只能上传20个文件。
