# CSS

# 一、CSS简介

CSS是层叠样子表的简称，也可以称为CSS样式表、级联样式表。

CSS也是一门标记语言。

CSS主要用于设置HTML页面中的文本内容（字体、大小、对齐方式等）、图片的外形、版面的布局和外观显示样式。

总：CSS可以美化html，让布局更简单。即结构（html)与样式（CSS)相分离。

语法规范：

<head>//写在head里
    <style>//样式标签
        //选择器{样式}
        p {//这里指给p标签修改样式
            color: red;
            font-size: 12px;
        }
    </style>
</head>

3、和结构分离了，但没完全分离，称为嵌入式引入。

适用于修改简单样式。

# 九、CSS的元素显示模式

html元素一般分成块元素和行内元素两种类型。

# 9.1 块元素

<h1>~<h6>、<p>、<div>、<ul>、<ol>、<li>等，其中div最典型。
1

特点：

1、独占一行

2、宽度、高度、外边距margin、内边距padding都可以控制。

3、宽度默认是父亲的宽度。

4、是一个容器盒子，可以放其它元素。

注：1、文字类元素内不能放块级元素。2、<p>里面不能放块级元素，特别是div。3、同理h1~h6也一样。

1

特点：

1、一行可以多个。

2、宽高直接设置无效。

3、默认宽度是它本身内容的宽度

4、行内元素只能容纳文本或其他行内元素。

注：1、链接里不能放链接。2、a可以放块级元素，但给a转换一下块级模式最安全

# 9.3行内块元素

<img/>、<input/>、<td>、<buttom>同时具有块元素和行内元素的特点，称之为行内块元素。

特点：

1、一行可以多个行内块元素，但之间会有空白缝隙。（行）

2、本身宽度为它内容的宽度。（行）

3、高度、行高、外边距及内边距都可以控制（块）

# 十、CSS背景

# 10.1背景颜色

background-color:transparent（默认）/color;

# 10.2背景图片

比插入图片便于控制位置。一般用于小图片或超大图片。

background-image: none（默认）/url(url地址)

图片比盒子小时，图片会被多次复制平铺开来，即背景平铺。

//获取网上的图片，检查-点击-选择对应的背景链接-open in new tab-保存

# 10.3背景平铺

background-repeat: repeat(默认)/no-repeat/repeat-x/repeat-y

不平铺时，少图片较小时，会像文字一样显示于左上角。

既可以加背景图片，也可以加颜色，不过颜色在最底层，图片会压住颜色，只有没铺图片的空隙才会显示颜色。

即从底到表为：背景颜色、背景图片、文字

# 10.4背景图片位置

background-position: x y;

x坐标和y坐标，可以使用方位名词或者精确单位。

参数值 说明
length 百分数/由浮点数字和单位标识符组成的长度值
xy即为距离左侧和上侧的距离。
只指定一个数值时，这个数值一定是x的，另一个默认是垂直居中。
position top/center/bottom/left/center/right 方位名词，方位名词xy坐标的顺序没关系，默认为左上
如果只指定了一个方位名词，另一个省略，则第二个默认居中对齐

混合单位：可以精确单位和方位名词混合使用，此时第一个值是x坐标，第二个是y坐标。

# 10.5 背景图像固定（背景附着）

background-attachment设置背景图像是否固定或随着页面的其余部分滚动。

background-attachment： scroll（默认，滚动）/fixed（固定）

# 10.6背景复合写法

background: 背景颜色 图片地址 平铺 滚动 位置；//习惯上这样写，实际上无顺序要求，也无必写要求。

复合写法也是提倡使用的。

# 10.7背景色半透明

css3提供了背景色半透明的效果。只有IE9+版本的浏览器才支持的。

background: rgba(0, 0, 0, 0.3);

最后一个参数为alpha透明度，取值范围在0~1之间。0：透明；1：不透明。如0.3习惯上把0省略，直接写.3。

小技巧：少有好几个一样的标签排成一列时，按shift，在要输入的地方全连越来，再输入就所有都有了。

# 十一、CSS的三大特性

有三个非常重要的特性：层叠性、继承性、优先级。

# 11.1层叠性

给相同的选择器设置了相同的样式，此时一个样式就会**覆盖（层叠）**另一个冲突的样式。层叠性主要解决样式冲突的问题

层叠性原则：就近原则，哪个离结构近，就执行哪个样式。

# 11.2继承性

css中的继承：子标签会继承父标签的某些样式，如文本颜色和字号。

点击检查便能看到：inherited from div从div中继承。

可以简化代码，降低css的复杂性。

子元素可以继承父元素的样子（text-,font-,line-这些元素开头的可以继承，以及color属性）也就是文字相关的样式。

//行高可以跟px单位的，也可以跟没有单位的，如1.5，是指当前元素文字大小的1.5倍，如被继承后，也就变成了子元素文字大小的1.5倍

# 11.3优先级

当同一个元素指定多个选择器，就会有优先级的产生。

  • 选择器相同，则执行层叠性
  • 选择器不同，是根据选择器权重执行
选择器 选择器权重
继承或* 0，0，0，0
元素选择器（标签） 0，0，0，1
类选择器，伪类选择器 0，0，1，0
id选择器 0，1，0，0
行内样式style="" 1，0，0，0
!important重要的 无穷大
div{
	color: pink!important;
}
1
2
3

注：

1、权重是有4组数字组成，永远不会有进位。即类选择器永远大于元素选择器，id选择器永远大于类选择器。

2、可简单记为，第一个为0.第二个为1，第三个为100，以此类推。

3、a链接浏览器默认指定了一个样式：蓝色的，有下划线。所以一般在body里指定颜色对a不起作用，因为继承的权重是0。

# 11.4权重的叠加

复合选择器会有权重叠加的问题。如0001+0001=0002。

权重会叠加，但不会有进位。

例：a:hover——0，0，1，1

# 十二 盒子模型

网页布局的三大核心 ：盒子模型、浮动和定位。

# 12.1盒子模型的四大组成部分

1、网页元素基本都是盒子Box

2、设置好盒子样式和位置

3、往盒子里装内容

盒子模型：把html页面的布局元素看作是一个矩形的盒子。

盒子包括：边框、外边距、内边距、和实际内容。

# 12.2 边框（border)

border: border-width ||border-style || border-color

属性 作用
border-width 一般单位px
border-style 样式（默认none,solid实线，dashed虚线，dotted点线）
border-color 颜色

可复写，可分开写。没顺序要求。

边框可分开写：如

border-top:1px solid red;//只设置上边框
1

表格的细线边框

border-collapse:collapse;合并相临的边框

边框会影响盒子的实际大小。=盒子+边框。

# 12.4内边距（paddind)

padding-left/right/top/bottom

复合属性：

值的个数 意思
padding:5px 1个值，即表示上下左右都是5px
padding:5px 10px 2个，分别是（上下），（左右）
padding:5px 10px 20px 3个，分别是上，（左右），下
padding:5px 10px 20px 30px 4个值，分别是上右下左，顺时钟

也就是全都是顺时钟，如果没有设置的边则与对边一样。

padding,也会影响盒子的实际大小。即盒子大小=盒子原大小（内容大小）+内边距+边框。

当div之类的块元素不设置宽度时，它默认继承父亲的宽度。所以此时设置padding,它的左右宽度不会受影响，只会挤占内容的宽度，高度就会变大。屏幕宽度为一般为1300px时。而当设置了宽度，如100%(或1300px)时，它的左右宽度也会变大，会超出父窗口的宽度，当超出屏幕时会出现滚动条。当不设置height属性也是一样的。

# 12.5外边距（margin)

margin-top/right/bottom/left

简写与padding一致

//盒子的可视化边距之类的，也能在检查里的，Computed里看到。

外边距可以让块级盒子水平居中，但有2个条件：//

2、盒子的左右外边距设计为auto(自动)，它便会水平居中显示

# 12.6外边距合并

只会出现在垂直情况，左右不出现的。

相邻块垂直外边距合并

以大的为准。

解决办法，给一个盒子增加margin就行了。

嵌套块元素垂直外边距的塌陷

对两个嵌套关系（父子关系）的块元素，父元素有上外边距同时子元素也有上外边距，此时父元素会塌陷较大的外边距值。

解决方案：

1、可以为父元素定义上边框（可以理解为，没定义边框之前，子元素无法找到父元素的上边，都是以最外边的屏幕边框为准，所以此时两个属性设置都变成了成屏幕边框的距离，哪个大以哪个为准。而父子元素因父元素没设置边框，所以之间无间距）

2、可以为父元素定义内边距。

3、可以为父元素添加overflow:hidden.//暂时记住后面再教，也不需要加边框使盒子变大，推荐使用。

# 12.10盒子阴影

box-shadow为盒子添加阴影

box-shadow:h-shadow v-shadow blur spread color inset;
1
值 描述
h-shadow 必需，水平阴影的位置，允许负值。水平垂直阴影位置都为0时，阴影和盒子位置重合，只会根据阴影的尺寸四周可能有小部分阴影显示。当水平为正时，阴影右移，负时，阴影左移多少距离。
v-shadow 必，垂直阴影的位置，允许负值。正为向下，负为向上。
blur 可选，模糊的距离。影子的虚实。0是纯实的影子，数字越大越模糊。
spread 可，阴影的尺寸。比正常盒子大多少或少多少。
color 可，阴影的颜色
inset 可，将外部阴影（outset)改为内部阴影；
box-shadow: 10px 10px 10px -4px rgb(0,0,0,.3);
1

盒子阴影不占用空间，不会影响其它盒子的排列。
